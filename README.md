# Homebrew Tap

This is a personal Homebrew Tap for custom tools and scripts.

## Installation

To use this tap, add it to your Homebrew:

```shell
brew tap Marcel2603/tap
```

After adding the tap, you can install the desired tool:

```shell
brew install <tool-name>
```

# Updates

The tools in this tap are automatically updated via GitHub Actions whenever a new release is published in the source
repository.To ensure you have the latest versions, run:

```shell
hbrew update
brew upgrade
```
